Списки

Список определений

Позволяет создавать списки терминов с их разъяснением

<dl> — Definitions List;
<dt> — Description Term;
<dd> — Description Details;
Нумерованный список
Позволяет нумеровать элементы текста. Это нужно для перечисления в строгом порядке.
Маркированный список
Позволяет перечислять элементы списка в любом порядке.
Список
Позволяет перечислять понятия и давать им опредления.

Нумерованный список

<ol> —Ordered List;
<li> — List Item;
  1. List Item 1
  2. List Item 2
  3. List Item 3
  4. List Item 4
  5. List Item 5
